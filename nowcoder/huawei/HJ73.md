## 描述

根据输入的日期，计算是这一年的第几天。

保证年份为4位数且日期合法。

进阶：时间复杂度：O(n)\*O*(*n*) ，空间复杂度：O(1)\*O*(1) 

### 输入描述：

输入一行，每行空格分割，分别是年，月，日

### 输出描述：

输出是这一年的第几天

## 示例1

输入：

```
2012 12 31
```

复制

输出：

```
366
```

复制

## 示例2

输入：

```
1982 3 4
```

复制

输出：

```
63
```

复制
