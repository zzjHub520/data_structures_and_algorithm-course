## 描述

合并 k 个升序的链表并将结果作为一个升序的链表返回其头节点。

数据范围：节点总数满足 0 \le n \le 10^50≤*n*≤105，链表个数满足 1 \le k \le 10^5 \1≤*k*≤105 ，每个链表的长度满足 1 \le len \le 200 \1≤*l**e**n*≤200 ，每个节点的值满足 |val| <= 1000∣*v**a**l*∣<=1000

要求：时间复杂度 O(nlogk)*O*(*n**l**o**g**k*)

## 示例1

输入：

```
[{1,2,3},{4,5,6,7}]
```

复制

返回值：

```
{1,2,3,4,5,6,7}
```

复制

复制


复制
